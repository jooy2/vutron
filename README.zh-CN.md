**Vutron** 是一个预配置的模板，用于开发 `Electron` 跨平台桌面应用。它使用 `Vue 3`，使您能够轻松构建快速的开发环境。

## 使用优势

- ✅ 无需任何预设，即可立即构建，快速开发。
- ✅ 快速维护，与最新的 `Vue` 和 `Electron` 以及许多模块兼容。
- ✅ 通过使用各种附加模板，无需担心布局和数据管理。

## 特点

- ⚡️ 通过热重载实现快速开发
- ⚡️ 跨平台开发和构建支持
- ⚡️ 支持自动化应用程序测试
- ⚡️ 支持 TypeScript
- ⚡️ 多语言支持
- ⚡️ 支持主题（暗色和亮色）
- ⚡️ 基本布局管理器
- ⚡️ 通过 Pinia 存储进行全局状态管理
- ⚡️ 通过 GitHub 社区和官方文档提供快速支持
